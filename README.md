# Snake Game React

A modern implementation of the classic Snake Game built with React and Tailwind CSS.

## 🎮 About

This project recreates the classic Snake Game using React. The goal is to control the snake, collect food, increase the score, and survive as long as possible without colliding with walls or the snake's own body.

This project is being built to strengthen React development skills, state management, keyboard event handling, and game logic implementation.

## ✨ Features

- Snake movement using keyboard controls
- Food generation
- Score tracking
- Game over detection
- Restart game functionality
- High score persistence with Local Storage
- Responsive design
- Modern UI with Tailwind CSS

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- Tailwind CSS
- Vite

## 🚀 Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
